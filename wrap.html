HTML → APK (Capacitor) Guide

Step-by-step wrapping HTML menjadi aplikasi Android.

1. Install Node.js

Download dan install Node.js LTS dari:

https://nodejs.org/

2. Masuk ke Folder Project

cd path\ke\project

3. Inisialisasi Project

npm init -y

4. Install Capacitor

npm install @capacitor/core @capacitor/cli @capacitor/android

5. Inisialisasi Capacitor

npx cap init

Contoh:

  • App Name: My App
  • Package ID: com.nama.myapp

6. Tambahkan Android

npx cap add android

7. Struktur Folder

project/
├── www/
│   ├── index.html
│   ├── style.css
│   └── script.js
├── android/
├── capacitor.config.json
└── package.json

Jika memakai Vite, ubah webDir menjadi dist.

8. Sync

npx cap sync

9. Buka Android Studio

npx cap open android

10. Build APK

  1. Build
  2. Build APK(s) atau Generate Signed Bundle / APK
  3. Tunggu hingga muncul "BUILD SUCCESSFUL".

FAQ

Internet?
Ya. Aplikasi tetap bisa menggunakan fetch(), REST API, Cloudflare Workers, Firebase, Supabase, dan layanan online lainnya.

Edit HTML?
Edit file web → jalankan npx cap sync → build APK lagi.

Ganti icon?
Gunakan Android Studio → New → Image Asset.
